How to run program:

If you are using an IDE:
1. Open files in IDE.
2. Compile and Run

If you are using command line:
1. Drag files into a directory
2. Use the command line command CD to change to directory holding files
3. Type 'make' and the program will compile
4. Type 'java ExamTester' to run the program
